
# 📧 CodeAlpha Python Internship - Email Extractor

### 📌 Description

This project is a simple Python application that extracts all valid email addresses from a text file using Regular Expressions (Regex). The extracted email addresses are then saved into a new text file.

---

## ✨ Features

- Reads data from a text file
- Extracts valid email addresses
- Removes duplicate email addresses
- Saves the extracted emails into a new file
- Displays the total number of emails found

---

## 🛠 Technologies Used

- Python 3
- Regular Expressions (`re`)
- File Handling

---




---

## 📥 Sample Input (emails.txt)

```
Hello,

Contact us at support@gmail.com

For business:
sales@company.com

admin@yahoo.com
```

---

## 📤 Sample Output (extracted_emails.txt)

```
support@gmail.com
sales@company.com
admin@yahoo.com
```
